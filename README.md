# Dashboard de Vendas
Criar um Dashboard de Vendas, com Excel.
## Descrição do Desafio
Construir um Dashboard de Vendas, interativo, com objetivo de organizar, analisar e vizualizar, de forma estratégica, od dados relacionados à Vendas de Games Xbox.
## Aprendizado
Tabelas dinâmicas, Gráficos, Segmentação de Dados, Layout etc.
## Intruções de Uso
Faça o download do arquivo disponível neste repositório;
Abra o arquivo .xslx, no Excel;
Use os filtros de segmentação para visualizar os dados desejados.
